Minecraft Server erstellen: eine Einführung

von Benjamin Hartmann

Installiere den server

|

\/

Lade die aktuelle Server-Datei herunter

Hier geht es zur offiziellen Website von Minecraft

Drücke dann auf minecraft_server.X.XX.X um die Datei herunterzuladen

Kopiere die server.jar in ein neuer Verzeichnis, zum Beispiel mcserver.

Benenne die Datei in "minecraft_server um"

Erstelle eine neue Textdatei namens "start.bat", in die du nun schreibst: 

						@echo off
						java -Xmx1024M -Xms1024M -jar minecraft_server.jar
						pause

Das @echo off wird verwendet, um die Ausgabe zu verschönern

Der pause Befehl sorgt dafür, dass wenn der Server crasht, man zuerst [ENTER] drücken muss, um das Fenster zu schließen

der Java Befehl wird verwendet um den Server zu starten

Führe die start.bat nun aus, ein paar Ordner und Dateien werden erstellt und ...

sie wird abstürzen

Öffne nun die neue Datei eula.txt und ändere die Zeile eula=false in eula=true. Damit akzeptierst du die Endbenutzerlizenzvereinbarung von Minecraft-Servern

Starte nun die Datei neu

Nun passiert etwas mehr:

Den neu installierten Server starten

Führe start.bat aus, um deinen Server zu starten, 2 Fenster wergen sich öffnen:

  • ein schwarz-weißes mit Text und
  • eines mit 3 abteilungen

schreibe nun in das schwarze "op [DEIN SPIELERNAME IN MINECRAFT]"

starte Minecraft

klicke auf Mehrspieler

Die Serveradresse ist "localhost";
dieser Server wird nur bei dir funktionieren
